Place your app icons here before building:

  icon.ico   — Windows (256×256 recommended, ICO format)
  icon.icns  — macOS  (1024×1024 recommended, ICNS format)
  icon.png   — Linux / tray icon (256×256 PNG)

Free tools:
  - https://cloudconvert.com/png-to-ico   (PNG → ICO)
  - https://cloudconvert.com/png-to-icns  (PNG → ICNS)

If you skip this step electron-builder will use its default icon.
The tray icon will also appear blank — replace icon.png to fix that.
